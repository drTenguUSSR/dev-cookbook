# 04-1.dm-fix

## Проверка активного/установленных DM

```bash
systemctl status display-manager 2>/dev/null || echo "Display Manager не активен"
```

## Список установленных пакетов DM

```bash
dpkg -l | grep -E "^(ii|rc)" | grep -E "(lightdm|gdm3|sddm|xdm|lxdm)"
```

## Полная зачистка

```bash
sudo apt purge lightdm
sudo systemctl set-default multi-user.target 
sudo reboot
```

## Восстановление пакета x11-xsession (рекомендуется)

```bash
# 1. Переустановка критических пакетов
sudo apt update
sudo apt install --reinstall -y x11-common xinit dbus-x11
sudo apt install -y xfce4 xfce4-goodies xrdp xorgxrdp

# 2. Восстановление конфигурации Xsession.d
sudo apt install -y --reinstall x11-common

# 3. Проверка восстановления
ls -la /etc/X11/Xsession /etc/X11/Xsession.d/ 2>/dev/null | head -10

# 4. Перезапуск служб
sudo systemctl restart xrdp xrdp-sesman
```
